# نَسَّق | NASSAQ - Product Requirements Document

## نظرة عامة
نظام إدارة مدارس متكامل يعمل بالذكاء الاصطناعي، متعدد المستأجرين (Multi-tenant)

## المستخدمون
1. **مدير المنصة (Platform Admin)** - إدارة جميع المدارس والمستخدمين
2. **مدير المدرسة (School Principal)** - إدارة مدرسة واحدة
3. **المعلم (Teacher)** - إدارة الفصول والحصص
4. **الطالب (Student)** - عرض الجدول والدرجات
5. **ولي الأمر (Parent)** - متابعة أداء الأبناء

---

## الميزات المُنجزة

### مركز القيادة (Command Center) ✅
- أزرار موحدة: "طالب / طلاب"، "معلم / معلمين"، "فصل / فصول"
- كروت مؤشرات الأداء (الطلاب، المعلمين، الفصول) قابلة للنقر
- قسم "يحتاج تدخل الآن" للمشكلات العاجلة
- نسبة الحضور اليوم (المعلمين والطلاب)

### الجدول المدرسي ✅
- زر "معالجة الجدول بالذكاء الاصطناعي" (بدلاً من "توليد")
- إزالة زر "+جديد"
- تكامل مع إعدادات المدرسة

### إدارة المستخدمين والفصول ✅
- أزرار موحدة مع مركز القيادة
- فلتر تلقائي عند النقر من الكروت
- عرض ديناميكي للبيانات

### إدارة حضور المعلمين والاداريين ✅
- تغيير الاسم من "إدارة الحضور"
- تقارير نسب الحضور الفعلية

### مركز التواصل والاشعارات ✅
- تغيير "الرسائل" إلى "الرسائل المستلمة"
- كروت قابلة للنقر لعرض التفاصيل

### التقارير والتحليلات ✅
- إلغاء كارت "متوسط الدرجات"
- بيانات ديناميكية من قاعدة البيانات

### رؤى الذكاء الاصطناعي ✅
- تحسين ألوان النص في كارت "مؤشر الأداء الذكي"

### إعدادات الحساب ✅
- إضافة قائمة الألقاب (السيد، دكتور، أستاذ...)
- زر تبديل الدور بين الحسابات

### بوابة الطالب ✅
- لوحة تحكم مع الترحيب والإحصائيات
- صفحة الجدول الدراسي
- صفحة الدرجات
- صفحة الحضور والغياب
- صفحة الواجبات

### بوابة ولي الأمر ✅
- لوحة تحكم مع قائمة الأبناء
- صفحة تفاصيل الابن
- عرض الدرجات والحضور والجدول

### تنظيف البيانات الوهمية ✅
- حذف 7 مستخدمين تجريبيين (@test.com)
- حذف 2 مدارس اختبارية
- تنظيف مجموعات demo_*

### دمج الاستيراد/التصدير في إدارة المستخدمين ✅ (مارس 2026)
- نقل وظائف صفحة BulkImportExportPage إلى صفحة UsersClassesManagement
- إضافة تبويب "استيراد/تصدير" مع تبويبات فرعية للاستيراد والتصدير
- الاستيراد: دعم الطلاب والمعلمين، تحميل القوالب، رفع الملفات، عرض النتائج
- التصدير: دعم 5 أنواع (الطلاب، المعلمين، الجدول، الحضور، الدرجات)
- إزالة الصفحة المستقلة ورابطها من القائمة الجانبية
- إزالة مسار /admin/bulk من التوجيه

### إصلاح بطاقات إحصائيات إدارة المستخدمين ✅ (مارس 2026)
- إصلاح مشكلة عرض أرقام قديمة/خاطئة في بطاقات الإحصائيات
- إزالة الحساب المحلي للإحصائيات من fetchUsers() الذي كان يُلغي بيانات الـ API
- الاعتماد بالكامل على `/api/admin/command-center/stats` للإحصائيات الحية
- جميع البطاقات تعكس الآن البيانات الحقيقية من قاعدة البيانات
