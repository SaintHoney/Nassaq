- **اسم المشروع**: نَسَّق | NASSAQ

## التقنيات المستخدمة
- **Backend**: FastAPI, Python, MongoDB (Motor), Pydantic
- **Frontend**: React.js, TailwindCSS, Shadcn/UI, Recharts
- **المصادقة**: JWT-based Authentication
- **الأمان**: RBAC (Role-Based Access Control), Multi-Tenant Isolation
- **التدقيق**: Audit Engine للتتبع والمراجعة
